503 Brant St, Burlington

Open-source web map for the final GIS project

Site overview

This map focuses on 503 Brant Street in downtown Burlington, Ontario. The subject site is shown with a highlighted marker and circle, while surrounding zoning, road segments, and nearby building footprints provide local planning context.

Project concept

This web map is designed to support a mixed-use redevelopment discussion by showing the site within its downtown context. It mirrors the ArcGIS Online version using an open-source Leaflet workflow.
